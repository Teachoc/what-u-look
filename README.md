# what-u-look
# 追番用项目：你瞅啥

## 功能
- 可以添加番剧到监视列表，并在更新后自动把状态重置到"本周未观看"
- 部署方式见下文
- 正在加入非 B 站番剧，以及美剧的抓取
- **不要**构建出 production 包，因为跨域问题用了 vue 的内置代理
